## PART 8: COMMON ISSUES & FIXES
## ═══════════════════════════════════════════════════════════

### ISSUE 1: 404 Error on Dashboard
CAUSE: No profile created yet
FIX: Create profile first (Profile tab)

### ISSUE 2: "Cannot connect to backend"
CAUSE: Backend not running or wrong IP
FIX: 
  1. Check backend terminal - should say "Application startup complete"
  2. Check mobile/config.js has correct IP
  3. Phone and laptop on same WiFi

### ISSUE 3: "Module not found: emergentintegrations"
CAUSE: AI library not installed
FIX: pip install emergentintegrations --extra-index-url https://d33sy5i8bnduwe.cloudfront.net/simple/

### ISSUE 4: Port already in use
FIX:
  # Backend (8001):
  lsof -ti:8001 | xargs kill -9
  
  # Expo (8081):
  lsof -ti:8081 | xargs kill -9

### ISSUE 5: Expo keeps showing errors
FIX:
  # Stop Expo (Ctrl+C)
  # Clear cache and restart:
  npx expo start --clear

### ISSUE 6: MongoDB not starting
FIX:
  # Check if running:
  docker ps
  
  # Start if stopped:
  docker start mongodb
  
  # Create new if doesn't exist:
  docker run -d -p 27017:27017 --name mongodb mongo:latest
