## 🎯 What You Need to Do

### Required (Must Do)
1. **Click "Save to GitHub"** in Emergent
2. **Pull code** on your laptop: `git pull origin main`
3. **Install dependencies:**
   - Mobile: `cd mobile && npm install`
   - Backend: `cd backend && pip install -r requirements.txt`
4. **Update IP address** in `mobile/config.js` (find your IP with `ipconfig`)
5. **Start services** (MongoDB, Backend, Expo)

### Optional (If You Want AI Nutritionist)
1. Add balance to your Emergent LLM key
2. Go to: Emergent → Profile → Universal Key → Add Balance
3. Add at least $5 for comfortable testing

## 💡 Pro Tips

1. **Keep backend terminal open** - The backend needs to stay running
2. **Use same Wi-Fi** - Phone and laptop must be on the same network
3. **First load takes time** - Wait 30-60 seconds for Expo to bundle the app
4. **Create profile first** - Go to Profile tab before using Dashboard
5. **Check backend URL** - Visit `http://YOUR_IP:8001/docs` to verify backend is running

## 🆘 Need Help?

### Quick Fixes

**"Network request failed"**
- Make sure backend is running
- Check IP in `config.js` is correct
- Phone and laptop on same Wi-Fi

**"Create Your Profile First"**
- This is normal! Just go to Profile tab

**AI showing "Budget exceeded"**
- Add balance to your LLM key (instructions in `AI_FEATURES_STATUS.md`)

**Can't find my IP address**
```powershell
ipconfig
```
Look for `IPv4 Address` (usually `192.168.x.x`)
